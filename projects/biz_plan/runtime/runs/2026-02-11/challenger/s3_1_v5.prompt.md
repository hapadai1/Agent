5) 차별화 강제(DIFFERENTIATION_WEAK 방지, 위반 시 실패)
═══════════════════════════════════════════════════════════════
차별화는 기능이 아니라 수익·원가·운영 관점으로만 서술하며, 아래 3개 중 최소 2개를 반드시 포함하고 각 항목에 근거 [R#]를 연결하라.
삭제 요청 대행의 변동비를 건당 과금으로 회수하는 구조(어떤 비용이 어떤 과금으로 상쇄되는지).[R1]
VIP 구성의 공급자 유형(보험/법률/관제/상담)과 제공 범위 및 면책/제한 조건의 구조(외부 1차 출처로 확인 가능한 범위만 단정).[R5][R1]
케이스 단위 운영(탐지→증거→삭제요청→종결)과 과금/원가 단위의 일치 방식.[R1]
위 3개 중 외부 출처로 확인 가능한 범위가 제한될 경우, 확인 가능한 범위만 단정하고, 확인 불가 항목은 확인 불가로 명시하며 구체 약관/상한 수치는 생성 금지.[R1]
경쟁/대안 대비 우위·차별화 표현(더 좋다/낫다/유리/불리 포함)은 비교표(D)에 최소 2개 축 이상으로 고정하고, 수치가 없으면 확인 불가를 사용하되 우리 서비스 축은 R1 기반으로 운영·정산·로그 관점의 차이를 명확히 기술하라.[R1]

6) VAGUE_CLAIMS 완전 차단(위반 시 실패)
═══════════════════════════════════════════════════════════════
절대 금지 표현: 획기적/대폭/명백히/확실히 및 동의어, 유망, 대중적, 자리잡았다.
조건부 금지 표현: 성장가능/확장/확보/경쟁력/효율/효과/가능/잠재/핵심/중요/높다/낮다/추세/심화 및 동의어.
위 조건부 금지 표현을 사용하려면 같은 문장 안에 반드시 (대상/기간/표본) + 수치 + [R#]를 포함.
모든 수치(금액/비율/건수/기간/시간/성장률/한도/보장/면책 조건의 수치)는 해당 문장 끝에 즉시 [R#] 표기(표는 해당 셀에 [R#]).
가능, 필요, 개선, 증가, 감소, 최적, 최소, 최대 등의 방향성 표현은 (1) 측정 지표 정의 (2) 데이터 원천 (3) 보고 주기 중 최소 2개 이상을 같은 문단에 포함해 검증 가능하게 고정하라.[R1]

7) 근거(출처) 규칙 및 참고문헌 규칙(위반 시 실패)
═══════════════════════════════════════════════════════════════
본문은 [R#]만 사용(문장 끝), 본문에 URL/문서 링크 직접 표기 금지.
본문에서 근거를 설명하기 위한 기관명/문서명 서술 금지(출처 표시는 [R#]로만 처리).
외부 근거 [R] 최소 3개 이상 포함 필수(내부 사업계획 [R1] 포함 가능하나, 외부 1차 출처 3개 이상을 반드시 확보해 반영).
외부 근거는 반드시 본문 주장에 직접 연결되어야 하며, 단순 나열 금지.
원문 확인 없이 그럴듯한 수치/약관/상한/보장 내용을 생성 금지.
외부 출처는 아래 제공된 외부 1차 출처 패키지 내에서만 사용 가능하며, 새로운 URL을 추가하거나 임의로 변경하는 행위는 금지한다.

8) 최종 출력 지시
═══════════════════════════════════════════════════════════════
이제 위 입력값을 바탕으로 3-1. 비즈니스 모델 및 수익 구조 섹션 본문 + [참고문헌]만 출력하라.

9) 자동 검수 규칙(출력에 포함하지 말고, 작성 중 내부적으로만 적용)
═══════════════════════════════════════════════════════════════
FORMAT_NONCOMPLIANCE 방지: 첫 줄 정확성, 2블록 제한, 표 파이프 규칙(모든 행 |로 시작·종료), 표 외 파이프 금지, 탭 0개, 본문 URL 0개, 본문 따옴표 0개, [참고문헌] 한 줄 항목 규칙, 마지막 줄 종결 문자 준수.
NO_EVIDENCE_OR_CITATION 방지: 모든 수치 문장 끝 [R#] 부착, 표 내 수치 셀 [R#] 부착, 2024.10 관측/계획/가정과 As-of 확인 불가 구분, R2~R5 중 최소 3개를 본문 주장에 실제 연결.
DIFFERENTIATION_WEAK 방지: 3개 차별화 항목 중 최소 2개를 수익·원가·운영 관점으로 명시하고 [R#] 연결, 경쟁/대안 비교표(D)에 최소 5개 대안 포함, 우위 표현은 표(D)와 근거 또는 확인 불가로 고정.
VAGUE_CLAIMS 방지: 금지어 사용 여부 점검, 조건부 금지어는 동일 문장 내 (대상/기간/표본)+수치+[R#] 없으면 삭제, 산업 평균 단정 금지, 확인 불가를 수치로 대체하지 말 것.

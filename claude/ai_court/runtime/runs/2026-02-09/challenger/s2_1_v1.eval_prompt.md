2-1. 창업아이템 현황 (준비 정도)

개발·추진 경과(이력)
본 과제는 AI 기반 법원 경매 물건 분석 및 투자 추천 솔루션을 목표로 하며, 2024년 3월에 개발을 착수하였다.[R1] 개발 착수 이후 경매 물건 분석에 필요한 입력 데이터 정의, 문서 해석 결과의 표준 출력 형식, 투자 판단 근거의 구조화 규칙을 내부 산출물로 정리하여 개발 기준으로 적용하였다.[R1] 2024년 8월 베타 버전의 MVP를 출시하여 외부 사용자 검증을 위한 운영 체계를 가동하였다.[R2] 베타 운영 과정에서 수집된 분석 결과와 사용자 피드백을 기반으로 성능 지표와 기능 동작을 동일 기준으로 재측정할 수 있도록 검증 절차를 정리하였다.[R3]

개발/개선 단계(TRL) 및 기능 범위
사업 신청 시점의 개발 단계는 TRL 6이며, 시스템·서비스 시제품 시연 수준으로 확인하였다.[R5] TRL 6의 근거는 베타 MVP를 기반으로 핵심 기능 흐름을 시제품 형태로 구성하고, 실제 시연 기록을 통해 동작을 검증한 결과에 따른다.[R5] 현재 기능 범위는 사건 단위 경매 물건 분석 결과를 구조화하여 제시하고, 투자 의사결정에 필요한 근거를 표준화된 형태로 제공하는 흐름을 포함한다.[R2] 베타 테스트 기준 분석 정확도는 78%로 측정되어, 시연 단계에서의 성능 수준을 수치로 확인하였다.[R3]

목표시장(고객) 반응
베타 테스트는 127명의 테스터 참여로 운영되었으며, 참여자 기반의 사용성 검증 데이터를 확보하였다.[R3] 베타 테스터 만족도는 5.0점 만점 기준 4.2점으로 측정되어, 서비스 사용 경험에 대한 정량 반응을 확보하였다.[R3] 베타 기간 누적 분석 건수는 3,420건으로 집계되어, 실제 사용 맥락에서의 처리량과 반복 사용 데이터를 축적하였다.[R3] 파일럿 고객 운영 과정에서는 분석 결과의 제시 방식과 업무 적용 흐름에 대한 피드백을 수집하여, 개선 항목을 기능 정의에 반영하는 방식으로 정리하였다.[R4]

주요 사업화 성과
현재까지 파일럿 고객은 부동산 중개업체 3곳을 확보하여, 베타 MVP의 적용 가능성을 고객 환경에서 확인하였다.[R4] 기술 검증 성과로 한국프롭테크포럼 기술 인증을 2024.10에 완료하여, 외부 기관 기준의 기술 검증 결과를 확보하였다.[R6] 또한 TRL 6 수준의 시스템·서비스 시제품 시연 근거를 갖추어, 사업화 이전 단계에서의 개발 준비 정도를 단계 지표로 제시할 수 있는 상태를 마련하였다.[R5]

잔여 과제 및 협약기간 내 고도화 계획(준비 정도 관점)
현재까지 확보한 베타 지표와 파일럿 운영 결과를 기준으로, 잔여 과제는 분석 정확도 측정 결과에 대한 오류 유형 정리, 결과 근거 제시의 일관성 강화, 사용자 흐름에서의 입력 데이터 누락 대응 체계 정비로 정리되어 있다.[R3][R4] 협약기간 내 고도화 계획은 베타 기간에 축적된 누적 분석 데이터와 운영 피드백을 기반으로, 동일한 측정 기준에서 성능을 반복 검증하고 개선 항목의 반영 여부를 추적 가능하도록 개발 산출물과 검증 기록을 연동하는 방향으로 정의되어 있다.[R3][R5] 이를 통해 TRL 6 시연 수준에서 확인된 기능 흐름을 기준으로, 고객 적용 과정에서 발생하는 예외 케이스 처리와 결과 품질 관리 절차를 문서화하여 재현 가능한 운영 체계로 확장하는 준비를 진행한다.[R4][R5]

수사/정성 위주 문장 금지. “우수/혁신/강력/차별적/경쟁력/효율/효과/가능/잠재/핵심/중요/높다/낮다/확장” 등의 표현은 동일 문장에 (대상/기간/표본) + 수치 + [R#]가 없으면 사용 금지.
“~할 수 있다/가능하다/기대된다/할 예정이다” 문장은 동일 문장에 숫자 + [R#]가 없으면 삭제하고, 정의·산식·측정계획 문장으로 치환하라.

6) 가격체계 타당성(필수 포함, 단 근거 없으면 ‘정의+검증계획’만)
────────────────────────────────────────
“수익모델(가격체계) 및 과금단위” 소제목 내에 반드시 아래 2가지를 포함하라.

(1) 가격 포지셔닝 검증 프레임: 경쟁/대체재 가격 벤치마크(월정액·건당·리포트 단가 등) 비교 항목(최소 3항목)과 수집경로·기간·정의(동일 조건) 제시. 단, 비교대상/가격 수치는 추가 근거가 있을 때만 기재.[R#]

(2) 사용자 지불의사 검증 프레임: 유료 의사 확인 지표(결제, 예약, 선결제, 견적요청, 유료 대기 등록 등 중 최소 2개)와 산식, 원천(로그/문서), 표본·기간을 정의. 수치 결과는 추가 근거가 있을 때만 기재.[R#]

────────────────────────────────────────
7) 외부 검증 가능한 사업화 성과(신뢰 지표) 작성 규칙(필수)
────────────────────────────────────────
“외부 검증 가능한 사업화 성과(신뢰 지표)” 소제목에는 최소 3개 지표를 포함하고, 각 지표마다 아래 4요소를 반드시 포함하라.

무엇(지표 정의)

산식/판정 기준(문서 존재 여부, 전환 이벤트, 서명/발신 확인 등)

원천(내부 로그/외부 문서 구분)

표본·기간(수치 임의 생성 금지: 근거 없으면 ‘협약기간 내’ 수준으로만 정의)

예시 지표 유형(이름만 참고, 실제 수치·기관명·문서명은 추가 근거가 있을 때만): LOI/MOU, 기관/전문가 파일럿(PoC), 유료 대기 리스트, 유료 의사 확인(결제/예약/견적요청) 이벤트.[R#]

────────────────────────────────────────
8) 구독형 단위경제(획득·유지·수익성 지표: CAC/LTV/리텐션) 작성 규칙(필수)
────────────────────────────────────────
“구독형 단위경제(획득·유지·수익성 지표: CAC/LTV/리텐션)” 소제목에는 최소 3개 지표를 포함하라(아래 중 3개 이상).

CAC(원/유료고객) = (채널별 마케팅비 + 판매비)/신규 유료 고객 수

ARPA 또는 ARPU(원/월) = 월 구독 매출/유료 고객 수

리텐션(%) = 특정 코호트의 t개월 후 유료 유지 고객 수/코호트 유료 고객 수

월 이탈률(%) = 해당 월 유료 해지 고객 수/해당 월 시작 시점 유료 고객 수

LTV(원) = ARPA × 평균 유료 유지개월(또는 ARPA/월 이탈률 등, 단 선택 산식은 명시)

각 지표마다 “정의/산식”과 “측정방법/원천(결제·갱신·해지 로그, 환불 로그, 마케팅비 집계, 채널 태깅 로그 등)”과 “표본·기간”을 명시하라.
수치 목표/수치 결과는 추가 근거가 있을 때만 기재하고, 없으면 산식·원천·기간만 정의하라.[R#]

────────────────────────────────────────
9) 계산(산식) 규칙(오류 시 실패)
────────────────────────────────────────
계산값(예: 예상 유료 사용자 수, 예상 월 구독 매출)은 반드시 “500명, 15~20%, 29,000/59,000, 매출 0”만으로 산식과 결과를 제시하라.[R1][R2][R3][R4]
산식은 한 문장 안에서 (입력값 → 계산 → 결과)를 명확히 제시하고, 계산 결과는 정수로 표기하라.[R#]
계산 결과가 본문/KPI/경제적 효과에서 서로 불일치하면 실패로 간주한다.

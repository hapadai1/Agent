3-1. 비즈니스 모델 및 수익 구조

AI기반 마이디지털정보 오남용 알림 서비스의 수익 구조는 사용자가 노출 사실을 늦게 인지하고 채널별로 분산된 조치를 반복하는 문제를, 구독형 상시 탐지와 케이스 기반 처리로 전환하는 데서 출발한다. 사용자는 개인정보 및 사진·동영상이 유출 또는 재유통되는 경로를 일일이 추적하기 어렵고, 삭제 요청을 위해 증거를 모으고 진행을 추적하는 과정이 서비스 품질을 좌우하는 비용 항목으로 전환된다. 따라서 본 서비스는 (1) 탐지 범위와 대응 범위를 요금제로 구분하고 (2) 케이스 처리 단위를 과금 단위와 일치시키며 (3) 변동비가 큰 항목은 건당 과금 또는 B2B 정산 구조로 회수하는 방식으로 손익 구조를 설계한다.

3-1-1. 구독 기반 수익모델(요금제·가치교환·과금 단위·업셀 경로)
구독 매출은 베이직, 프리미엄, VIP 3개 요금제로 구성하며, 가치교환은 탐지 범위, 조치 대행 범위, 긴급 대응 및 보장 범위로 구분한다.[R1] 과금 단위는 개인 사용자 월 구독료이며, 동일 사용자가 상위 요금제로 이동할 때 탐지 채널과 대응 옵션이 확장되도록 설계해 업셀의 기준을 기능 나열이 아니라 운영 단위 변화로 정의한다.[R1] 요금제 믹스에 따른 ARPU는 ARPU = (베이직 월요금 x 베이직 유료가입자 수 + 프리미엄 월요금 x 프리미엄 유료가입자 수 + VIP 월요금 x VIP 유료가입자 수) / 총 유료가입자 수로 정의하며, ARPU는 월별 유료 결제 기준으로 집계하고 환불·미수금은 결제 정산 기준에 따라 차감한다.[R1]

구독 모델의 운영 원칙은 케이스 단위로 탐지→증거→삭제요청→종결을 연결하고, 각 단계에서 발생하는 변동비 항목이 어느 요금제의 과금 단위로 회수되는지 명시하는 것이다. 베이직은 다채널 감지의 변동비를 월 구독료에 포함시키고, 프리미엄은 삭제 요청 대행을 포함하되 초과분을 건당 과금으로 전환해 변동비 급증 구간을 요금제 밖으로 분리한다.[R1] VIP는 공급자 유형이 상담(법률·관제·심리·사기 대응)과 보험으로 구성되며, 보험 보장은 약관상 손해 및 복구 비용 등 특정 항목으로 한정되고 지급 요건이 전제되는 구조를 반영해, 보험료와 상담비를 사용자·월 변동비로 관리한다.[R2][R3]

3-1-2. 추가 수익원(B2B 제휴·기업 계약·부가서비스 과금·정산 구조)
추가 수익원은 통신사 부가서비스 제휴, 기업용 임직원 보호 연간 계약, 그리고 삭제 요청 대행 건당 과금(프리미엄 초과분)으로 구성한다.[R1] 통신사 제휴의 계약 단위는 통신사 채널의 부가서비스 SKU 단위이며, 월 정산액은 부가서비스 판매가 x 유효가입자 수 x 수익배분율로 정의하고, 고객지원·환불·미수금 처리 규칙을 정산 조항으로 고정해 월별 수익 인식과 현금흐름을 분리 관리한다. 기업 계약의 계약 단위는 연간 계약이며, 과금 단위는 보호 대상 임직원 수 또는 임직원 계정 수를 기준으로 하고, ARPA는 ARPA = 연간 계약금액 / 계약 고객사 수, ARPPU(기업)는 연간 계약금액 / 유료 보호 대상자 수로 정의해 단가와 침투율을 분리 추적한다.

삭제 요청 대행 건당 과금은 원가-과금 정합성 관점에서 차별화 지점으로 작동한다. 삭제 요청은 플랫폼별 정책 확인, 증거 패키징, 요청 제출, 반려 대응, 종결 확인까지 반복 작업이 포함되어 인력·법무·제휴 수수료 등 변동비가 케이스 수에 비례한다. 본 구조는 프리미엄 기본 제공 범위를 넘어서는 초과 케이스를 건당 과금으로 전환해, 삭제 요청 대행의 건당 변동비를 동일한 건당 매출로 상쇄하도록 설계한다.[R1] 해외 동종 구독 서비스의 월 구독료가 월 $12 또는 월 $24.99 수준으로 제시되는 사례가 확인되며, 일부 요금제는 신원도용 보험 및 복구 지원을 포함한다.[R2][R4]

3-1-3. 목표시장 진입 현황 및 사업화 추진성과(파일럿/테스터/출시 일정/다음 단계)
현재 단계는 기획·개발 단계로 매출이 없으며, 파일럿 테스터 25명과 만족도 4.3/5.0이 2024.10 기준으로 확보되어 있다.[R1] 정식 서비스 출시는 2024.10 기준 로드맵에서 2025년 Q2로 계획되어 있으며, 베타에서 유료 전환과 케이스 처리 원가를 측정해 요금제 믹스와 변동비 회수 구조를 확정하는 것을 사업화의 관문 지표로 둔다.[R1] 외부 공개 집계에서 2024년 데이터 침해 이벤트 3,158건과 피해 통지 1,350,835,988건이 보고된 바 있어, 개인이 사후 인지 지연을 겪는 환경이 지속적으로 형성된다.[R7]

다음 의사결정 포인트는 (1) 무료체험 종료 시점과 결제 전환의 타이밍, (2) 삭제 요청 대행의 기본 제공량과 초과 과금의 경계값, (3) VIP에서 보험·상담 제공의 공급 단가와 면책 조건의 운영 가능성으로 정의한다. 무료체험→유료 전환율은 개발 단계에서 내부 사업계획상 15-20%를 참고 범위로 두되, 유사 구독 데이터에서 무료체험 전환이 7일 구간에 집중되고 B2C 7일 전환율이 15-18%로 관측된 점을 고려해, 체험 기간과 결제 유도 시점을 7일 단위로 실험 설계한다.[R1][R5] 삭제 요청 대행은 케이스 처리량이 원가의 직접 동인이므로, 베타에서 케이스당 투입 시간과 외부 수수료를 분해 계측해 건당 과금 단가의 원가 상쇄 여부를 검증한다.[R1]

3-1-4. 단위경제 및 손익 구조(원가 구성·총마진·CAC/LTV/Payback·손익분기, 단위경제 요약표 포함)
원가 구조는 변동비와 고정비로 구분한다. 변동비는 탐지 연산 및 크롤링, 저장 및 증거 보관, 외부 모니터링 연동, 삭제 요청 대행 처리 인력·법무 검토, 보험료(가입자 수 연동), 제휴 수수료(통신사 정산)로 구성되며, 요금제별로 변동비 항목이 다르게 배치된다.[R1] 고정비는 제품 개발 인력, 모델 고도화 및 품질 관리, 보안·컴플라이언스, 고객지원 운영 체계, 제휴 계약 관리 등으로 구성하고, 손익분기는 유료가입자 기여이익 기반으로 산정한다.

총마진은 총마진율 = (매출 - 변동비) / 매출로 정의하고, 월별로 결제 정산 매출과 변동비 실집계를 매칭해 산출한다.[R1] CAC는 CAC = (마케팅비 + 세일즈비 + 프로모션비) / 신규 유료가입자 수로 정의하며, 통신사 제휴는 채널 수수료와 공동 마케팅 분담금을 CAC에 포함하고, 자연 유입은 콘텐츠 제작비와 운영 인력을 포함해 채널별 CAC를 분리한다. LTV는 LTV = ARPU x 총마진율 / 이탈률로 정의하고, 이탈률은 월 이탈률 = 당월 해지 유료가입자 수 / 월초 유료가입자 수로 정의하며, Payback은 Payback(개월) = CAC / (월 ARPU x 총마진율)로 정의해 cohort 기준으로 측정한다.

손익분기 유료가입자 수는 N_BE = 월 고정비 / (월 ARPU x 총마진율)로 산정하고, 삭제 요청 대행의 케이스 단위 손익분기는 C_BE = 케이스 고정운영비 / (건당 과금 - 건당 변동비)로 산정해 케이스 운영이 전체 손익을 왜곡하지 않도록 분리 관리한다.[R1] 구독형 벤치마크에서 연간 이탈률 1-5% 또는 월 4% 수준이 기준점으로 제시되며, 이는 LTV 산식의 분모에 직접 반영되는 외부 기준 범위로 사용한다.[R6] 또한 구독형 소프트웨어 벤치마크에서 CAC Payback 중앙값 20개월, 상위 사분위 14개월이 제시되어, 베타 이후 채널 믹스와 요금제 믹스 확정 시 Payback 목표 범위를 수립하는 근거로 사용한다.[R8]

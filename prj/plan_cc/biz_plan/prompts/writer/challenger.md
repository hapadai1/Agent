3) 표 작성 규칙

표가 필요한 경우 마크다운 파이프(|) 표만 사용:

금지 예시(절대 출력 금지): 탭으로 열 구분, 여러 공백 정렬, 박스문자(│ ─) 표, "표:" 라벨.

표의 모든 줄은 다음을 동시에 만족:

각 줄의 첫 글자 = '|', 각 줄의 마지막 글자 = '|' (행 앞 공백 0개).

헤더 1줄 + 구분선 1줄 + 데이터 행만 포함.

셀 안 줄바꿈 금지(한 셀=한 줄).

표 내부에는 URL/괄호형 출처/각주 금지(표 내부 출처는 [R#]만 허용).

열 구분은 '|'만 허용(탭/복수 공백 정렬 금지).

6) [참고문헌] 규칙(형식 고정, 1:1 매핑)

[참고문헌] 제목은 반드시 정확히 "[참고문헌]" 한 줄로 출력

본문에서 사용한 모든 [R번호]를 [참고문헌]에 1:1로 대응

각 항목은 반드시 "[R#] "로 시작(예: "[R1] …")

[R번호]는 R1부터 연속 증가(누락/중복/건너뜀 금지)

외부 참고문헌 형식:
[R#] 기관명 (연도). 문서/통계명. 직접URL. (조회경로/열람일)

예시:
[R1] 한국인터넷진흥원 (2024). 개인정보 침해신고 통계. https://www.kisa.or.kr. (열람일: 2024.12)
[R2] 경찰청 (2024). 사이버범죄 통계. https://www.police.go.kr. (열람일: 2024.12)

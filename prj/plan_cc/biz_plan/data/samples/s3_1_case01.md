# 입력 조건

## 주제
AI기반 마이디지털정보 오남용 알림 서비스

## 서비스 개요
개인정보 및 사진/동영상 보호 AI 서비스 구독 모델. 내가 모르는 사이에 유출되거나 노출된 개인정보(사진, 영상 포함)를 AI가 자동으로 탐지하고 알려주는 서비스.

## 작성 목적/독자
- 신규 IT 서비스 기획서 (투자자/의사결정자 관점)
- 톤: 제출용 완성 문서

## 필수 포함 항목 (체크리스트)
- [ ] 수익모델 (비즈니스 모델) 상세 기술
- [ ] 목표시장(고객) 진입 현황
- [ ] 사업화 추진성과 (가입자, 테스터 등)
- [ ] 성과 지표 표

## 제공 근거 (사용 가능 팩트) ★ 출처+URL 표기 필수 ★

### 수익 모델 (월정액 구독)
- 베이직 (기본 보호): 4,900원/월
  - 내 사진·영상 유출 감지 (SNS + 웹)
  - 개인정보 유출 감지 (전화번호, 이메일)
  - 기본 경고 알림

- 프리미엄 (심층 보호): 9,900원/월
  - 딥웹 & 다크웹 개인정보 유출 감지
  - 게시물 삭제 요청 대행 서비스
  - AI 안전 필터 기능

- VIP (긴급 대응 포함): 19,900원/월
  - 법적 대응 지원 (전문가 상담)
  - 사이버 보험 포함
  - 24시간 긴급 대응 서비스

### 추가 수익원
- B2B: 통신사 부가서비스 제휴 (수익 배분)
- B2B: 기업용 임직원 보호 서비스 (연간 계약)
- 부가서비스: 삭제 요청 대행 건당 과금 (프리미엄 초과분)

### 현재 성과 (기획/개발 단계)
- 파일럿 테스터: 25명 (2024.10 기준)
- 테스터 만족도: 4.3/5.0
- 정식 서비스 출시 예정: 2025년 Q2
- 아직 매출 없음 (개발 단계)

### 예상 전환율
- 무료 체험 → 유료 전환율: 15-20% (SaaS 업계 평균 참고)
